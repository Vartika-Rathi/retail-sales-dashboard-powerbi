# 📊 US Retail Sales Dashboard — Power BI

An interactive business intelligence dashboard analyzing 9,800+ retail transactions across the US from 2015–2018.

## 📌 Key Insights
- Technology is the highest revenue category ($836K in sales)
- New York City leads all cities in total sales
- Sales peak consistently in Q4 each year

## 🛠️ Tools Used
- Power BI Desktop
- Microsoft Excel (data cleaning)
- Dataset: Kaggle Superstore Sales Dataset
